# Global Pack Studio

**Автоматизированная система для управления производственными процессами**

## Описание

Global Pack Studio — коммерческий пет‑проект, направленный на автоматизацию рутинных задач в производственной среде.

Система решает следующие задачи:
* планирование производства;
* учёт продукции;
* сканирование продукции (в разработке);
* расчёт затрат (в разработке).

Ключевая особенность — личный кабинет работника с дифференцированным функционалом в зависимости от роли пользователя.

## Технологический стек

* **Фронтенд:** React, Vue.js.
* **Управление состоянием:** Redux (хранилище данных), Context (управление состояниями).
* **Бэкенд:** Nest.js.
* **База данных:** PostgreSQL.
* **Контейнеризация:** Docker, Docker Compose.
* **Деплой:** GitHub Actions + Yandex Cloud.
* **Дизайн:** две темы интерфейса (светлая и тёмная).

## Запуск проекта

1. Перейдите в папку `backend` и выполните:
   ```bash
   npm run start

2. Перейдите в папку `frontend` и выполните:
   ```bash
   npm run dev

## Системные требования

- Node.js (актуальная версия);
- PostgreSQL (настроенный сервер);
- Docker и Docker Compose (для контейнеризации);
- доступ к Yandex Cloud (для деплоя);
- GitHub Account (для использования GitHub Actions).

## Планы по доработке

В активной разработке находятся следующие модули:

- система сканирования продукции — внедрение механизма автоматического считывания данных с продукции;
- модуль расчёта затрат — разработка алгоритма калькуляции себестоимости и операционных расходов.
